# PriorityIQ — Feature Prioritization Tool

A RICE-score based feature prioritization tool for product managers. Rank features by Reach, Impact, Confidence, and Effort to make data-driven roadmap decisions.

## Live Demo
[feature-prioritization-tool.vercel.app](https://feature-prioritization-tool.vercel.app)

## Features
- RICE scoring framework (Reach × Impact × Confidence ÷ Effort)
- Auto-ranked feature list with priority badges
- Priority Matrix chart (Impact vs Effort bubble chart)
- Category filtering and search
- Export to CSV
- Persistent storage via localStorage

## Tech Stack
React, Recharts, Vite, Vercel
